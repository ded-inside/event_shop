# event_shop

## Description
Web application of the event shop
## Installation env (for vscode)

``` bash
git clone https://github.com/ded-inside/event_shop/
cd event_shop
python -m venv .venv
cd .venv/Scripts
activate # (for windows)
source activate # (for linux)
cd ../..
```

## Installation app
``` bash
pip install -r requirements.txt
```

## Usage
``` bash
flask run
```
